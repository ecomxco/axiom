# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — 2026-03-08

### Added

- **15 deterministic workflows** for AI-agent-driven software development
- **Project Initialization:** `/map-codebase`, `/plan-project`
- **Phase Execution:** `/discuss-phase`, `/research-phase`, `/plan-phase`, `/analyze-gaps`, `/execute-phase`, `/verify-work`, `/diagnose-issues`
- **Session Management:** `/context`, `/progress`, `/quick`, `/pause-work`, `/resume-project`, `/clear`
- **8-type test matrix** (unit, integration, regression, contract, math, boundary, security, UAT)
- **4-rule deviation system** for handling unplanned work during execution
- **Goal-backward verification** with conversational UAT and natural language severity inference
- **9-check gap analysis** safety net between planning and execution
- **Scientific debugging** methodology with hypothesis → evidence → root cause
- **Session persistence** — UAT state survives across session clears
- **Multi-agent compatibility** — works with Antigravity, Claude, Cursor, ChatGPT, Gemini
